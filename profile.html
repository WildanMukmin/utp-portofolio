Tentang Saya

Saya adalah mahasiswa S1 Ilmu Komputer di UNILA dengan keahlian dalam pengembangan web full-stack. Saya mengkhususkan diri dalam penggunaan teknologi modern seperti React, Next.js, Laravel, dan SQL. Salah satu proyek penting saya adalah mengembangkan sistem web pusat karier secara mandiri, yang menunjukkan kemampuan saya dalam menangani siklus pengembangan penuh.

Selain keahlian teknis saya, saya memiliki lebih dari satu tahun pengalaman sebagai desainer grafis lepas untuk Capcut, yang membuat templat video inovatif, templat grafis, dan desain stiker. Saya juga menjabat sebagai asisten mahasiswa, yang berkontribusi dengan mengelola tugas entri data dan membuat desain yang berdampak untuk berbagai kebutuhan.

Tujuan saya adalah untuk lebih memajukan karier saya sebagai pengembang full-stack atau mengkhususkan diri sebagai pengembang frontend atau backend. Keterampilan desain saya meningkatkan kemampuan saya untuk membuat aplikasi web yang menarik dan berpusat pada pengguna, yang mendukung hasrat saya untuk pengembangan dan desain.
